24 # DESCRIPTION
25 
26 Control Utility for the ce\_gw Kernel Programm.
27 
28 
29 
30 # OPTIONS
31 
32 - __\-b__, __\--bidirectional__
33 
34  If you add a route from SRC to DST, there will be also add the same route back from DST to SRC.
35 
36 - __\-f__, __\--can-fd__
37 
38  A Flag for the larger CAN-FD Frames.
39 
40 - __\-t__ _TYPE_, __\--type__=_TYPE_
41 
42  Set The Type of the Gateway. '__net__' is the Default. __udp__, __tcp__ are not supported yet.
43 
44 - _TYPE_ := { __none__ | __eth__ | __net__ | __udp__ | __tcp__ } Types
45 
46 # COMMANDS
47 
48 - __add__ __route__ _SRC_ _DST_
49 
50  Add a Gateway from _SRC_ to _DST_, where SRC and DST are the String names of the Interfaces.
51 
52 - __add__ __dev__
53 
54  Add a virtual ethernet device. You need this device for adding a route. With the options \[ __\-f__ | __\--can-fd__ \] \[ __\-t__ _TYPE_ | __\--type__=_TYPE_ \] you can set a meaningfull MTU.
55 
56  +------+-----+------+--------------------------+
57  | TYPE | FD? | BYTE | MTU Size Description |
58  +------+-----+------+--------------------------+
59  | NONE | --- | 1500 | Standart Ethernet MTU |
60  | ETH | NO | 8 | Max CAN Payload |
61  | ETH | YES | 64 | Max CAN-FD Payload |
62  | NET | NO | 16 | Max CAN Frame |
63  | NET | YES | 72 | Max CAN-FD Frame |
64  | UDP | --- | 1500 | Standart Ethernet MTU |
65  | TCP | --- | 1500 | Standart Ethernet MTU |
66  +------+-----+------+--------------------------+
67 
68 - __del__ __route__ _ID_
69 
70  Deletes the Gateway with _ID_.
71 
72 - __del__ __dev__ _NAME_
73 
74  Deletes a previously with __add__ __dev__ added device with _NAME_ as name.
75 
76 - __route__ \[_ID_\]
77 
78  List active Gateways and Informations or if _ID_ is specified, Information of the Gateway with _ID_ will printed
79
