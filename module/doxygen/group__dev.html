struct net_device * ce_gw_dev_alloc ( void  )
read

Allocates a Ethernet Device for the Gateway.

Allocates a Ethernet Device with struct ce_gw_job_info as private data. ce_gw_dev_setup() should be called after this function. free_netdev(dev) musst be called for freeing the memory.

Return values
NULLIf an error has occured.
Returns
A pointer to the allocated device.
